## :clipboard: Documentation
### Frontend

```
cd client
npx create-react-app ./
npm install axios moment react-file-base64 redux redux-thunk
npm start
```

### Backend

```
cd server
npm init -y
npm install body-parser cors express mongoose nodemon
```
